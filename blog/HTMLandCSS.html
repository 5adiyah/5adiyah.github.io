HTML & CSS

Jan 12, 2016

We’re in week four of the Epicodus Evening Intro class. We’ve been learning Html and CSS. The last couple of weeks have been pretty slow because I’ve taken HTML and CSS before but it was a good refresher.

Our first code review was to create a landing page for our web programming portfolio. Some of the criteas included:

  • Use of all the HTML tags we’ve learned: p, h, ul, ol, li, em, strong, a, img, div, span
  • Use Boostrap classes to style the page
  • Commits made regularly
    I didn’t end up using Bootstrap. I played around with it for a bit before deciding that I didn’t like it. I found Skeleton which I loved. It was pretty lightweight and super easy to figure out.

Another thing that I struggled with was committing regularly. I’m told that it will eventually become second nature but right now it feels more like a hassle. I keep forgetting to commit until I’m about an hour deep into the project.
